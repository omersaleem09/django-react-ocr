# django-react-ocr

**clone the repo**

**DJANGO**
-------------------------------
1. cd taxgpt
2. create virtual env
3. pip install -r requirements.txt
4. python manage.py migrate
5. python manage.py createsuperuser admin
6. python manage.py runserver

**REACT**
------------------------------------
1. cd ocr-react-frontend
2. npm install
3. npm start
